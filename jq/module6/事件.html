# 5. 事件

# 5.1 blur

概述： 当元素失去焦点时触发 blur 事件。

示例如下：

  1. 触发所有段落的blur事件
$("p").blur();
  1. 任何段落失去焦点时弹出一个 "Hello World!"在每一个匹配元素的blur事件中绑定的处理函数
$("p").blur( function () { alert("Hello World!"); } );

# 5.2 change

概述： 当元素的值发生改变时，会发生 change 事件。

示例如下：

  1. 触发被选元素的 change 事件
$(selector).change();
  1. 给所有的文本框增加输入验证
$("input[type='text']").change( function() {
  // 这里可以写些验证代码
});

# 5.3 click

概述： 触发每一个匹配元素的click事件。

示例如下：

  1. 触发页面内所有段落的点击事件
$("p").click();
  1. 将页面内所有段落点击后隐藏
$("p").click( function () { $(this).hide(); });

# 5.4 dblclick

概述： 当双击元素时，会发生 dblclick 事件。

示例如下：

  1. 给页面上每个段落的双击事件绑上 "Hello World!" 警告框
$("p").dblclick( function () { alert("Hello World!"); });

# 5.5 error

概述： 当元素遇到错误（没有正确载入）时，发生 error 事件。

示例如下：

  1. 在服务器端记录JavaScript错误日志
$(window).error(function(msg, url, line){
  jQuery.post("js_error_log.php", { msg: msg, url: url, line: line });
});
  1. 隐藏JavaScript错误
$(window).error(function(){
  return true;
});
  1. 给你IE的用户隐藏无效的图像
$("img").error(function(){
  $(this).hide();
});

# 5.6 focus

概述： 当元素获得焦点时，触发 focus 事件。

示例如下：

  1. 当页面加载后将 id 为 'login' 的元素设置焦点
$(document).ready(function(){
  $("#login").focus();
});
  1. 使人无法使用文本框
$("input[type=text]").focus(function(){
  this.blur();
});

# 5.7 focusin

概述： 当元素获得焦点时，触发 focusin 事件。

示例如下：

html代码

<p><input type="text" /> <span>focusout fire</span></p>
<p><input type="password" /> <span>focusout fire</span></p>
  1. 获得焦点后会触发动画
$("p").focusin(function() {
  $(this).find("span").css('display','inline').fadeOut(1000);
});

# 5.8 focusout

概述： 当元素获得焦点时，触发 focusin 事件。

示例如下：

html代码

<p><input type="text" /> <span>focusout fire</span></p>
<p><input type="password" /> <span>focusout fire</span></p>
  1. 获得焦点后会触发动画
$("p").focusout(function() {
  $(this).find("span").css('display','inline').fadeOut(1000);
});

# 5.9 keydown

概述： 当键盘或按钮被按下时，发生 keydown 事件。

示例如下：

  1. 在页面内对键盘按键做出回应，可以使用如下代码
$(window).keydown(function(event){
  switch(event.keyCode) {
    // ...
    // 不同的按键可以做不同的事情
    // 不同的浏览器的keycode不同
    // 更多详细信息:     http://unixpapa.com/js/key.html
    // ...
  }
});

# 5.10 keypress

概述： 当键盘或按钮被按下时，发生 keypress 事件。

示例如下：

  1. 计算在输入域中的按键次数
$("input").keydown(function(){  $("span").text(i+=1);});

# 5.11 keyup

概述： 当按钮被松开时，发生 keyup 事件。它发生在当前获得焦点的元素上。

示例如下：

  1. 当按下按键时，改变文本域的颜色
$("input").keyup(function(){
  $("input").css("background-color","#D6D6FF");
});

# 5.12 mousedown

概述： 当鼠标指针移动到元素上方，并按下鼠标按键时，会发生 mousedown 事件。

示例如下：

  1. 当按下鼠标按钮时，隐藏或显示元素
$("button").mousedown(function(){
  $("p").slideToggle();
});

# 5.13 mouseenter

概述： 当鼠标指针穿过元素时，会发生 mouseenter 事件。该事件大多数时候会与mouseleave 事件一起使用。

示例如下：

  1. 当鼠标指针进入（穿过）元素时，改变元素的背景色
$("p").mouseenter(function(){
  $("p").css("background-color","yellow");
});

# 5.14 mouseleave

概述： 鼠标指针离开元素时，会发生 mouseleave 事件。该事件大多数时候会与mouseenter 事件一起使用。

示例如下：

  1. 当鼠标指针离开元素时，改变元素的背景色
$("p").mouseleave(function(){
  $("p").css("background-color","#E9E9E4");
});

# 5.15 mousemove

概述： 当鼠标指针在指定的元素中移动时，就会发生 mousemove 事件。

示例如下：

  1. 获得鼠标指针在页面中的位置
$(document).mousemove(function(e){
  $("span").text(e.pageX + ", " + e.pageY);
});

# 5.16 mouseout

概述： 当鼠标指针从元素上移开时，发生 mouseout 事件。

示例如下：

  1. 当鼠标从元素上移开时，改变元素的背景色
$("p").mouseout(function(){
  $("p").css("background-color","#E9E9E4");
});

# 5.17 mouseover

概述： 当鼠标指针位于元素上方时，会发生 mouseover 事件。

示例如下：

  1. 当鼠标指针位于元素上方时时，改变元素的背景色
$("p").mouseover(function(){
  $("p").css("background-color","yellow");
});

# 5.18 mouseup

概述： 当在元素上放松鼠标按钮时，会发生 mouseup 事件。

示例如下：

  1. 当松开鼠标按钮时，隐藏或显示元素
$("button").mouseup(function(){
  $("p").slideToggle();
});

# 5.19 resize

概述： 当调整浏览器窗口的大小时，发生 resize 事件。

示例如下：

  1. 让人每次改变页面窗口的大小时很郁闷的方法
$(window).resize(function(){
  alert("Stop it!");
});
  1. 对浏览器窗口调整大小进行计数
$(window).resize(function() {
  $('span').text(x+=1);
});

# 5.20 scroll

概述： 当用户滚动指定的元素时，会发生 scroll 事件。

示例如下：

  1. 当页面滚动条变化时，执行的函数
$(window).scroll( function() { /* ...do something... */ } );
  1. 对元素滚动的次数进行计数
$("div").scroll(function() {
  $("span").text(x+=1);
});

# 5.21 select

概述： 当 textarea 或文本类型的 input 元素中的文本被选择时，会发生 select 事件。

示例如下：

  1. 触发所有input元素的select事件
$("input").select();
  1. 当文本框中文本被选中时执行的函数
$(":text").select( function () { /* ...do something... */ } );

# 5.22 submit

概述： 当提交表单时，会发生 submit 事件。

示例如下：

  1. 提交本页的第一个表单
$("form:first").submit();
  1. 如果你要阻止表单提交
$("form").submit( function () {
  return false;
} );

# 5.23 unload

概述： 在当用户离开页面时，会发生 unload 事件。

示例如下：

  1. 页面卸载的时候弹出一个警告框
$(window).unload( function () { alert("Bye now!"); } );
