# 3. 包裹

# 3.1 wrap

概述： 把所有匹配的元素用其他元素的结构化标记包裹起来。

示例如下：

  1. 把所有的段落用一个新创建的div包裹起来

html代码

<p>段落1</p><p>段落2</p>

jQuery代码

$("p").wrap("<div class='wrap'></div>");
  1. 用原先div的内容作为新div的class，并将每一个元素包裹起来

html代码

<div class="container">
  <div class="inner">Hello</div>
  <div class="inner">Goodbye</div>
</div>

jQuery代码

$('.inner').wrap(function() {
  return '<div class="' + $(this).text() + '" />';
});

# 3.2 unwrap

概述： 这个方法将移出元素的父元素。

示例如下：

html代码

<div>
    <p>Hello</p>
    <p>cruel</p>
    <p>World</p>
</div>
  1. 用ID是"content"的div将每一个段落包裹起来
$("p").unwrap()

# 3.3 wrapAll

概述： 将所有匹配的元素用单个元素包裹起来。

示例如下：

  1. 用一个生成的div将所有段落包裹起来
$("p").wrapAll("<div></div>");
  1. 用一个生成的div将所有段落包裹起来
$("p").wrapAll(document.createElement("div"));

# 3.4 wrapInner

概述： 获取第一个元素。

示例如下：

  1. 把所有段落内的每个子内容加粗
$("p").wrapInner("<b></b>");
  1. 把所有段落内的每个子内容加粗
$("p").wrapInner(document.createElement("b"));;
  1. 用原先div的内容作为新div的class，并将每一个元素包裹起来

html代码

<div class="container">
  <div class="inner">Hello</div>
  <div class="inner">Goodbye</div>
</div>

jQuery代码

$('.inner').wrapInner(function() {
  return '<div class="' + $(this).text() + '" />';
});
