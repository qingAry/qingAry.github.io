# 1. 内部插入

# 1.1 append

概述： 向每个匹配的元素内部追加内容。

示例如下：

html代码

<p>I would like to say: </p>
  1. 向所有段落中追加一些HTML标记
$("p").append("<b>Hello</b>");

# 1.2 appendTo

概述： 把所有匹配的元素追加到另一个指定的元素元素集合中。

示例如下：

  1. 把所有段落追加到ID值为foo的元素中

html代码

<p>I would like to say: </p>
<div id="foo"></div><div></div>

jQuery代码

$("p").appendTo("div#foo");
  1. 新建段落追加div中并加上一个class

html代码

<div></div><div></div>

jQuery代码

 $("<p/>")
   .appendTo("div")
   .addClass("test")
   .end()
   .addClass("test2");

# 1.3 prepend

概述： 向每个匹配的元素内部前置内容。

示例如下：

  1. 向所有段落中前置一些HTML标记代码

html代码

<p>I would like to say: </p>

jQuery代码

$("p").prepend("<b>Hello</b>");
  1. 将一个DOM元素前置入所有段落

html代码

<p>I would like to say: </p>
<p>I would like to say: </p>
<b class="foo">Hello</b>
<b class="foo">Good Bye</b>

jQuery代码

$("p").prepend( $(".foo")[0] );

# 1.4 prependTo

概述： 把所有匹配的元素前置到另一个、指定的元素元素集合中。

示例如下：

  1. 向所有段落中前置一些HTML标记代码

html代码

<p>I would like to say: </p><div id="foo"></div>

jQuery代码

$("p").prependTo("#foo");;
