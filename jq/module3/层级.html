# 2. 层级

# 2.1 ancestor descendant(层级)

概述： 在给定的祖先元素下匹配所有的后代元素。

示例如下：

html代码

<form>
  <label>Name:</label>
  <input name="name" />
  <fieldset>
      <label>Newsletter:</label>
      <input name="newsletter" />
 </fieldset>
</form>
<input name="none" />
  1. 找到表单中所有的 input 元素
$("form input")

# 2.2 parent > child

概述： 在给定的父元素下匹配所有的子元素。

示例如下：

html代码

<form>
  <label>Name:</label>
  <input name="name" />
  <fieldset>
      <label>Newsletter:</label>
      <input name="newsletter" />
 </fieldset>
</form>
<input name="none" />
  1. 匹配表单中所有的子级input元素
$("form > input")

# 2.3 prev + next

概述： 匹配所有紧接在 prev 元素后的 next 元素。

示例如下：

html代码

<form>
  <label>Name:</label>
  <input name="name" />
  <fieldset>
      <label>Newsletter:</label>
      <input name="newsletter" />
 </fieldset>
</form>
<input name="none" />
  1. 匹配所有跟在 label 后面的 input 元素
$("label + input")

# 2.4 prev ~ siblings

概述： 匹配 prev 元素的所有同辈 siblings 元素。

示例如下：

html代码

<form>
  <label>Name:</label>
  <input name="name" />
  <fieldset>
      <label>Newsletter:</label>
      <input name="newsletter" />
 </fieldset>
</form>
<input name="none" />
  1. 找到 form 元素的所有同辈 input 元素
$("form ~ input")
