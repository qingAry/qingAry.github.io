# 3. 基本伪类

# 3.1 :first

概述： 获取第一个元素。

示例如下：

html代码

<ul>
    <li>list item 1</li>
    <li>list item 2</li>
    <li>list item 3</li>
    <li>list item 4</li>
    <li>list item 5</li>
</ul>
  1. 获取匹配的第一个元素
$('li:first');

# 3.2 :not(selector)

概述： 去除所有与给定选择器匹配的元素。

示例如下：

html代码

<input name="apple" />
<input name="flower" checked="checked" />
  1. 查找所有未选中的 input 元素
$("input:not(:checked)")

# 3.3 :even

概述： 匹配所有索引值为偶数的元素，从 0 开始计数。

示例如下：

html代码

<table>
  <tr><td>Header 1</td></tr>
  <tr><td>Value 1</td></tr>
  <tr><td>Value 2</td></tr>
</table>
  1. 查找表格的1、3、5...行（即索引值0、2、4...）
$("tr:even")

# 3.4 :odd

概述： 匹配所有索引值为奇数的元素，从 0 开始计数。

示例如下：

html代码

<table>
  <tr><td>Header 1</td></tr>
  <tr><td>Value 1</td></tr>
  <tr><td>Value 2</td></tr>
</table>
  1. 查找表格的2、4、6行（即索引值1、3、5...）
$("tr:odd")

# 3.5 :eq(index)

概述： 匹配一个给定索引值的元素。

示例如下：

html代码

<table>
  <tr><td>Header 1</td></tr>
  <tr><td>Value 1</td></tr>
  <tr><td>Value 2</td></tr>
</table>
  1. 查找第二行
$("tr:eq(1)")

# 3.6 :gt(index)

概述： 匹配所有大于给定索引值的元素。

示例如下：

html代码

<table>
  <tr><td>Header 1</td></tr>
  <tr><td>Value 1</td></tr>
  <tr><td>Value 2</td></tr>
</table>
  1. 查找第二第三行，即索引值是1和2，也就是比0大
$("tr:gt(0)")

# 3.7 :lang1.9+

概述： 选择指定语言的所有元素。

示例如下：

  1. 选择所有 P 的语言属性
$("p:lang(it)")

# 3.8 :last

概述： 获取最后个元素。

示例如下：

html代码

<ul>
    <li>list item 1</li>
    <li>list item 2</li>
    <li>list item 3</li>
    <li>list item 4</li>
    <li>list item 5</li>
</ul>
  1. 获取匹配的最后个元素
$('li:last')

# 3.9 :lt(index)

概述： 匹配所有小于给定索引值的元素。

示例如下：

html代码

<table>
  <tr><td>Header 1</td></tr>
  <tr><td>Value 1</td></tr>
  <tr><td>Value 2</td></tr>
</table>
  1. 查找第一第二行，即索引值是0和1，也就是比2小
$("tr:lt(2)")

# 3.10 :header

概述： 匹配如 h1, h2, h3之类的标题元素。

示例如下：

html代码

<h1>Header 1</h1>
<p>Contents 1</p>
<h2>Header 2</h2>
<p>Contents 2</p>
  1. 给页面内所有标题加上背景色
$(":header").css("background", "#EEE");

# 3.11 :animated

概述： 匹配所有正在执行动画效果的元素。

示例如下：

html代码

<button id="run">Run</button><div></div>
  1. 匹配所有正在执行动画效果的元素
$("#run").click(function(){
  $("div:not(:animated)").animate({ left: "+=20" }, 1000);
});

# 3.12 :focus

概述： 匹配当前获取焦点的元素。

示例如下：

css代码

.focused {background: #abcdef;}

html代码

<div id="content">
  <input tabIndex="1"> 
  <input tabIndex="2">
  <select tabIndex="3">
    <option>select menu</option>
  </select>
  <div tabIndex="4"> a div</div>
</div>
  1. 添加一个"focused"的类名给那些有focus方法的元素
$( "#content" ).delegate( "*", "focus blur", function( event ) {
    var elem = $( this );
    setTimeout(function() {
       elem.toggleClass( "focused", elem.is( ":focus" ) );
    }, 0);
});

# 3.13 :root1.9+

概述： 选择该文档的根元素。

示例如下：

  1. 设置<html>背景颜色为黄色
$(":root").css("background-color","yellow");

# 3.14 :target1.9+

概述： 选择由文档URI的格式化识别码表示的目标元素。

示例如下：

$("p, button, h1, h2").click(function(event){
  $("div").html("Triggered by a " + event.target.nodeName + " element.");
});
