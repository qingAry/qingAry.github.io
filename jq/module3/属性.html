# 6. 属性

# 6.1 [attribute]

概述： 匹配包含给定属性的元素。注意，在jQuery 1.3中，前导的@符号已经被废除！如果想要兼容最新版本，只需要简单去掉@符号即可。

示例如下：

html代码

<div>
  <p>Hello!</p>
</div>
<div id="test2"></div>
  1. 查找所有含有 id 属性的 div 元素
$("div[id]")

# 6.2 [attribute=value]

概述： 匹配给定的属性是某个特定值的元素。

示例如下：

html代码

<input type="checkbox" name="newsletter" value="Hot Fuzz" />
<input type="checkbox" name="newsletter" value="Cold Fusion" />
<input type="checkbox" name="accept" value="Evil Plans" />
  1. 查找所有 name 属性是 newsletter 的 input 元素
$("input[name='newsletter']").attr("checked", true);

# 6.3 [attribute!=value]

概述： 匹配所有不含有指定的属性，或者属性不等于特定值的元素。

示例如下：

html代码

<input type="checkbox" name="newsletter" value="Hot Fuzz" />
<input type="checkbox" name="newsletter" value="Cold Fusion" />
<input type="checkbox" name="accept" value="Evil Plans" />
  1. 查找所有 name 属性不是 newsletter 的 input 元素
$("input[name!='newsletter']").attr("checked", true);

# 6.4 [attribute^=value]

概述： 匹配给定的属性是以某些值开始的元素。

示例如下：

html代码

<input name="newsletter" />
<input name="milkman" />
<input name="newsboy" />
  1. 查找所有 name 以 'news' 开始的 input 元素
$("input[name^='news']")

# 6.5 [attribute$=value]

概述： 匹配给定的属性是以某些值结尾的元素。

示例如下：

html代码

<input name="newsletter" />
<input name="milkman" />
<input name="jobletter" />
  1. 查找所有 name 以 'letter' 结尾的 input 元素
$("input[name$='letter']")

# 6.6 [attribute*=value]

概述： 匹配给定的属性是以包含某些值的元素。

示例如下：

html代码

<input name="man-news" />
<input name="milkman" />
<input name="letterman2" />
<input name="newmilk" />
  1. 查找所有 name 包含 'man' 的 input 元素
$("input[name*='man']")

# 6.7 [attrSel1][attrSel2][attrSelN]

概述： 复合属性选择器，需要同时满足多个条件时使用。

示例如下：

html代码

<input id="man-news" name="man-news" />
<input name="milkman" />
<input id="letterman" name="new-letterman" />
<input name="newmilk" />
  1. 找到所有含有 id 属性，并且它的 name 属性是以 man 结尾的
$("input[id][name$='man']")
