# 1. 基本

# 1.1 #id

概述： 根据给定的ID匹配一个元素。

示例如下：

  1. 查找 ID 为"myDiv"的元素

html代码：

<div id="notMe"><p>id="notMe"</p></div>
<div id="myDiv">id="myDiv"</div>

jQuery代码：

$("#myDiv");
  1. 查找含有特殊字符的元素

html代码：

<span id="foo:bar"></span>
<span id="foo[bar]"></span>
<span id="foo.bar"></span>

jQuery代码：

#foo\\:bar
#foo\\[bar\\]
#foo\\.bar

# 1.2 element

概述： 根据给定的元素名匹配所有元素。

示例如下：

html代码

<div>DIV1</div>
<div>DIV2</div>
<span>SPAN</span>
  1. 查找一个 DIV 元素
$("div");

# 1.3 .class

概述： 根据给定的类匹配元素。

示例如下：

html代码

<div class="notMe">div class="notMe"</div>
<div class="myClass">div class="myClass"</div>
<span class="myClass">span class="myClass"</span>
  1. 查找所有类是 "myClass" 的元素.
$(".myClass");

# 1.4 *

概述： 匹配所有元素。

示例如下：

html代码

<div>DIV</div>
<span>SPAN</span>
<p>P</p>
  1. 找到每一个元素
$("*")

# 1.5 selector1,selector2,selectorN

概述： 将每一个选择器匹配到的元素合并后一起返回。

示例如下：

html代码

<div>div</div>
<p class="myClass">p class="myClass"</p>
<span>span</span>
<p class="notMyClass">p class="notMyClass"</p>
  1. 找到匹配任意一个类的元素
$("div,span,p.myClass")
