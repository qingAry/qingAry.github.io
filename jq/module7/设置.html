# 5. 设置

# 5.1 jQuery.fx.off

概述： 关闭页面上所有的动画。

示例如下：

  1. 执行一个禁用的动画
  jQuery.fx.off = true;
  $("input").click(function(){
    $("div").toggle("slow");
  });

# 5.2 jQuery.fx.interval

概述： 设置动画的显示帧速。

示例如下：

  1. 帧速设置为100ms

html代码

<!DOCTYPE html>
<html>
<head>
  <style>
    div { width:50px; height:30px; margin:5px; float:left;
          background:green; }
    </style>
  <script src="http://code.jquery.com/jquery-1.5.2.js"></script>
</head>
<body>
  <p><input type="button" value="Run"/></p>
<div></div>
<script>
jQuery.fx.interval = 100;

$("input").click(function(){
  $("div").toggle( 3000 );
});
</script>

</body>
</html>
