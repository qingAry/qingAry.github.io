# 2. 位置

# 2.1 offset

概述： 获取匹配元素在当前视口的相对偏移。

示例如下：

html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第二段的偏移
var p = $("p:last");
var offset = p.offset();
p.html( "left: " + offset.left + ", top: " + offset.top );
  1. 设置第二段的偏移
$("p:last").offset({ top: 10, left: 30 });

# 2.2 position

概述： 获取匹配元素相对父元素的偏移。

示例如下：

html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第一段的偏移
var p = $("p:first");
var position = p.position();
$("p:last").html( "left: " + position.left + ", top: " + position.top );

# 2.3 scrollTop

概述： 获取匹配元素相对滚动条顶部的偏移。。

示例如下：

html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第一段相对滚动条顶部的偏移
var p = $("p:first");
$("p:last").text( "scrollTop:" + p.scrollTop() );
  1. 设置相对滚动条顶部的偏移
$("div.demo").scrollTop(300);

# 2.4 scrollLeft

概述： 获取匹配元素相对滚动条顶部的偏移。

示例如下：

html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第一段相对滚动条左侧的偏移
var p = $("p:first");
$("p:last").text( "scrollLeft:" + p.scrollLeft() );
  1. 设置相对滚动条左侧的偏移
$("div.demo").scrollLeft(300);
