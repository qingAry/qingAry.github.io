# 1. ajax 请求

# 1.1 $.ajax

概述： 通过 HTTP 请求加载远程数据。

示例如下：

  1. 加载并执行一个 JS 文件
$.ajax({
  type: "GET",
  url: "test.js",
  dataType: "script"
});
  1. 保存数据到服务器，成功时显示信息
$.ajax({
   type: "POST",
   url: "some.php",
   data: "name=John&location=Boston",
   success: function(msg){
     alert( "Data Saved: " + msg );
   }
});
  1. 装入一个 HTML 网页最新版本
$.ajax({
  url: "test.html",
  cache: false,
  success: function(html){
    $("#results").append(html);
  }
});
  1. 同步加载数据。发送请求时锁住浏览器。需要锁定用户交互操作时使用同步方式
 var html = $.ajax({
  url: "some.php",
  async: false
 }).responseText;
  1. 发送 XML 数据至服务器。设置 processData 选项为 false，防止自动转换数据格式
 var xmlDocument = [create xml document];
 $.ajax({
   url: "page.php",
   processData: false,
   data: xmlDocument,
   success: handleResponse
 });

# 1.2 load

概述： 载入远程 HTML 文件代码并插入至 DOM 中。

示例如下：

  1. 加载文章侧边栏导航部分至一个无序列表

html代码

<b>jQuery Links:</b>
<ul id="links"></ul>

jQuery代码

$("#links").load("/Main_Page #p-Getting-Started li");
  1. 加载 feeds.html 文件内容
$("#feeds").load("feeds.html");
  1. 同上，但是以 POST 形式发送附加参数并在成功时显示信息
 $("#feeds").load("feeds.php", {limit: 25}, function(){
   alert("The last 25 entries in the feed have been loaded");
 });

# 1.3 $.get

概述： 通过远程 HTTP GET 请求载入信息。

示例如下：

  1. 请求 test.php 网页，忽略返回值
$.get("test.php");
  1. 请求 test.php 网页，传送2个参数，忽略返回值
$.get("test.php", { name: "John", time: "2pm" } );
  1. 显示 test.php 返回值(HTML 或 XML，取决于返回值)。
$.get("test.php", function(data){
  alert("Data Loaded: " + data);
});
  1. 显示 test.cgi 返回值(HTML 或 XML，取决于返回值)，添加一组请求参数
$.get("test.cgi", { name: "John", time: "2pm" },
  function(data){
    alert("Data Loaded: " + data);
});

# 1.4 $.getJSON

概述： 通过 HTTP GET 请求载入 JSON 数据。

示例如下：

  1. 从 Flickr JSONP API 载入 4 张最新的关于猫的图片

html代码

<div id="images"></div>

jQuery代码

$.getJSON("http://api.flickr.com/services/feeds/photos_public.gne?tags=cat&tagmode=any&format
=json&jsoncallback=?", function(data){
  $.each(data.items, function(i,item){
    $("<img/>").attr("src", item.media.m).appendTo("#images");
    if ( i == 3 ) return false;
  });
});
  1. 从 test.js 载入 JSON 数据并显示 JSON 数据中一个 name 字段数据
$.getJSON("test.js", function(json){
  alert("JSON Data: " + json.users[3].name);
});
  1. 从 test.js 载入 JSON 数据，附加参数，显示 JSON 数据中一个 name 字段数据
$.getJSON("test.js", { name: "John", time: "2pm" }, function(json){
  alert("JSON Data: " + json.users[3].name);
});

# 1.5 $.getScript

概述： 通过 HTTP GET 请求载入并执行一个 JavaScript 文件。

示例如下：

  1. 载入<a title="http://jquery.com/plugins/project/color" class="external text" href="http://jquery.com/plugins/project/color">jQuery 官方颜色动画插件</a> 成功后绑定颜色变化动画

html代码

<button id="go">» Run</button>
<div class="block"></div>

jQuery代码

jQuery.getScript("http://dev.jquery.com/view/trunk/plugins/color/jquery.color.js", function(){
  $("#go").click(function(){
    $(".block").animate( { backgroundColor: 'pink' }, 1000)
      .animate( { backgroundColor: 'blue' }, 1000);
  });
});
  1. 加载并执行 test.js

jQuery代码

$.getScript("test.js");
  1. 加载并执行 test.js ，成功后显示信息

jQuery代码

$.getScript("test.js", function(){
  alert("Script loaded and executed.");
});

# 1.6 $.post

概述： 通过远程 HTTP POST 请求载入信息。

示例如下：

  1. 请求 test.php 网页，忽略返回值
$.post("test.php");
  1. 请求 test.php 页面，并一起发送一些额外的数据（同时仍然忽略返回值）：
$.post("test.php", { name: "John", time: "2pm" } );
  1. 向服务器传递数据数组（同时仍然忽略返回值）：
$.post("test.php", { 'choices[]': ["Jon", "Susan"] });
  1. 使用 ajax 请求发送表单数据
$.post("test.php", $("#testform").serialize());
  1. 输出来自请求页面 test.php 的结果（HTML 或 XML，取决于所返回的内容）：
$.post("test.php", function(data){
  alert("Data Loaded: " + data);
});
  1. 向页面 test.php 发送数据，并输出结果（HTML 或 XML，取决于所返回的内容）：
$.post("test.php", { name: "John", time: "2pm" },
  function(data){
    alert("Data Loaded: " + data);
});
  1. 获得 test.php 页面的内容，并存储为 XMLHttpResponse 对象，并通过 process() 这个 JavaScript 函数进行处理
$.post("test.php", { name: "John", time: "2pm" },
  function(data){
    process(data);
}, "xml");
  1. 获得 test.php 页面返回的 json 格式的内容：
$.post("test.php", { "func": "getNameAndTime" },
   function(data){
     alert(data.name); // John
     console.log(data.time); //  2pm
}, "json");
