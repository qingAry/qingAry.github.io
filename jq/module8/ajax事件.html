# 2. ajax 事件

# 2.1 ajaxComplete

概述： AJAX 请求完成时执行函数。Ajax 事件。

示例如下：

  1. AJAX 请求完成时执行函数
$("#msg").ajaxComplete(function(event,request, settings){
  $(this).append("<li>请求完成.</li>");
});
  1. 当 AJAX 请求正在进行时显示“正在加载”的指示
$("#txt").ajaxStart(function(){
  $("#wait").css("display","block");
});
$("#txt").ajaxComplete(function(){
  $("#wait").css("display","none");
});

# 2.2 ajaxError

概述： AJAX 请求发生错误时执行函数。Ajax 事件。

示例如下：

  1. AJAX 请求失败时显示信息
$("#msg").ajaxError(function(event,request, settings){
  $(this).append("<li>出错页面:" + settings.url + "</li>");
});

# 2.3 ajaxSend

概述： AJAX 请求发送前执行函数。Ajax 事件。

示例如下：

  1. AJAX 请求发送前显示信息
 $("#msg").ajaxSend(function(evt, request, settings){
   $(this).append("<li>开始请求: " + settings.url + "</li>");
 });

# 2.4 ajaxStart

概述： AJAX 请求开始时执行函数。Ajax 事件。

示例如下：

  1. AJAX 请求开始时显示信息
$("#loading").ajaxStart(function(){
  $(this).show();
});

# 2.5 ajaxStop

概述： AJAX 请求结束时执行函数。Ajax 事件。

示例如下：

  1. AJAX 请求结束后隐藏信息
 $("#loading").ajaxStop(function(){
   $(this).hide();
 });

# 2.6 ajaxSuccess

概述： AJAX 请求成功时执行函数。Ajax 事件。

示例如下：

  1. 当 AJAX 请求成功后显示消息
 $("#msg").ajaxSuccess(function(evt, request, settings){
   $(this).append("<li>请求成功!</li>");
});
