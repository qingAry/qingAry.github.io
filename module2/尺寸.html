# 3.尺寸

# 3.1 height

概述： 取得匹配元素当前计算的高度值（px）。

示例如下：

  1. 获取第一段的高
$("p").height();
  1. 把所有段落的高设为 20:
$("p").height(20);
  1. 以 10 像素的幅度增加 p 元素的高度
$("button").click(function(){
  $("p").height(function(n,c){
   return c+10;
  });
});

# 3.2 width

概述： 取得第一个匹配元素当前计算的宽度值（px）。

示例如下：

  1. 获取段落的宽度
$("p").width();
  1. 把所有段落的宽设为 20:
$("p").width(20);
  1. 以 10 像素的幅度增加 p 元素的宽度
$("button").click(function(){
  $("p").width(function(n,c){
    return c+10;
  });
});

# 3.3 innerHeight

概述： 获取第一个匹配元素内部区域高度（包括补白、不包括边框）。

示例如下：

html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第一段落内部区域高度。
var p = $("p:first");
$("p:last").text( "innerHeight:" + p.innerHeight() );

# 3.4 innerWidth

概述： 获取第一个匹配元素内部区域宽度（包括补白、不包括边框）。

示例如下：

html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第一段落内部区域宽度。
var p = $("p:first");
$("p:last").text( "innerWidth:" + p.innerWidth() );

# 3.5 outerHeight

概述： 获取第一个匹配元素外部高度（默认包括补白和边框）。

示例如下：

html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第一段落外部高度。
var p = $("p:first");
$("p:last").text( "outerHeight:" + p.outerHeight() + " , outerHeight(true):" + p.outerHeight(true) );

# 3.6 outerWidth

概述： 获取第一个匹配元素外部宽度（默认包括补白和边框）。

示例如下： html代码

<p>Hello</p><p>2nd Paragraph</p>
  1. 获取第一段落外部宽度。
var p = $("p:first");
$("p:last").text( "outerWidth:" + p.outerWidth() + " , outerWidth(true):" + p.outerWidth(true) );
