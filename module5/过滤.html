# 1. 过滤

# 1.1 eq

概述： 获取第N个元素。

示例如下：

  1. 获取匹配的第二个元素

html代码

<p> This is just a test.</p> <p> So is this</p>

jQuery代码

$("p").eq(1)
  1. 获取匹配的第二个元素

html代码

<p> This is just a test.</p> <p> So is this</p>

jQuery代码

$("p").eq(-2)

# 1.2 first

概述： 获取第一个元素。

示例如下：

html代码

<ul>
    <li>list item 1</li>
    <li>list item 2</li>
    <li>list item 3</li>
    <li>list item 4</li>
    <li>list item 5</li>
</ul>
  1. 获取匹配的第一个元素
$('li').first()

# 1.3 last

概述： 获取最后个元素。

示例如下：

html代码

<ul>
    <li>list item 1</li>
    <li>list item 2</li>
    <li>list item 3</li>
    <li>list item 4</li>
    <li>list item 5</li>
</ul>
  1. 获取匹配的最后个元素
$('li').last()

# 1.4 hasClass

概述： 检查当前的元素是否含有某个特定的类，如果有，则返回true。

示例如下：

html代码

<div class="protected"></div><div></div>
  1. 给包含有某个类的元素进行一个动画
$("div").click(function(){
  if ( $(this).hasClass("protected") )
    $(this)
      .animate({ left: -10 })
      .animate({ left: 10 })
      .animate({ left: -10 })
      .animate({ left: 10 })
      .animate({ left: 0 });
});

# 1.5 filter

概述： 筛选出与指定表达式匹配的元素集合。

示例如下：

  1. 保留带有select类的元素

html代码

<p>Hello</p><p>Hello Again</p><p class="selected">And Again</p>

jQuery代码

$("p").filter(".selected")
  1. 保留第一个以及带有select类的元素

html代码

<p>Hello</p><p>Hello Again</p><p class="selected">And Again</p>

jQuery代码

$("p").filter(".selected, :first")
  1. 保留子元素中不含有ol的元素

html代码

<p><ol><li>Hello</li></ol></p><p>How are you?</p>

jQuery代码

$("p").filter(function(index) {
  return $("ol", this).length == 0;
});

# 1.6 is

概述： 根据选择器、DOM元素或 jQuery 对象来检测匹配元素集合，如果其中至少有一个元素符合这个给定的表达式就返回true。

示例如下：

  1. 由于input元素的父元素是一个表单元素，所以返回true

html代码

<form><input type="checkbox" /></form>

jQuery代码

$("input[type='checkbox']").parent().is("form")
  1. 判断点击li标签增加背景色为红色，如果点击的是第2个strong,当前的li增加背景色为绿色

html代码

<ul>
  <li><strong>list</strong> item 1 - one strong tag</li>
  <li><strong>list</strong> item <strong>2</strong> - two <span>strong tags</span></li>
  <li>list item 3</li>
</ul>

jQuery代码

$("li").click(function() {
  var $li = $(this),
    isWithTwo = $li.is(function() {
      return $('strong', this).length === 2;
    });
  if ( isWithTwo ) {
    $li.css("background-color", "green");
  } else {
    $li.css("background-color", "red");
  }
});

# 1.7 map

概述： 将一组元素转换成其他数组（不论是否是元素数组）。

示例如下：

html代码

<p><b>Values: </b></p>
<form>
  <input type="text" name="name" value="John"/>
  <input type="text" name="password" value="password"/>
  <input type="text" name="url" value="http://ejohn.org/"/>
</form>
  1. 把form中的每个input元素的值建立一个列表
$("p").append( $("input").map(function(){
  return $(this).val();
}).get().join(", ") );

# 1.8 has

概述： 保留包含特定后代的元素，去掉那些不含有指定后代的元素。

示例如下：

html代码

<ul>
  <li>list item 1</li>
  <li>list item 2
    <ul>
      <li>list item 2-a</li>
      <li>list item 2-b</li>
    </ul>
  </li>
  <li>list item 3</li>
  <li>list item 4</li>
</ul>
  1. 给含有ul的li加上背景色
$('li').has('ul').css('background-color', 'red');

# 1.9 not

概述： 删除与指定表达式匹配的元素。

示例如下：

html代码

<p>Hello</p><p id="selected">Hello Again</p>
  1. 从p元素中删除带有 select 的ID的元素
$("p").not( $("#selected")[0] )

# 1.10 slice

概述： 选取一个匹配的子集。

示例如下：

  1. 选择第一个p元素

html代码

<p>Hello</p><p>cruel</p><p>World</p>

jQuery代码

$("p").slice(0, 1).wrapInner("<b></b>");
  1. 选择前两个p元素

html代码

<p>Hello</p><p>cruel</p><p>World</p>

jQuery代码

$("p").slice(0, 2).wrapInner("<b></b>");
  1. 只选取第二个p元素

html代码

<p>Hello</p><p>cruel</p><p>World</p>

jQuery代码

$("p").slice(1, 2).wrapInner("<b></b>");
  1. 只选取第二第三个p元素

html代码

<p>Hello</p><p>cruel</p><p>World</p>

jQuery代码

$("p").slice(1).wrapInner("<b></b>");
  1. 选取第最后一个p元素

html代码

<p>Hello</p><p>cruel</p><p>World</p>

jQuery代码

$("p").slice(-1).wrapInner("<b></b>");
