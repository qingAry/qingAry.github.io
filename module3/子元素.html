# 7. 子元素

# 7.1 :first-child

概述： 匹配第一个子元素。

示例如下：

html代码

<ul>
  <li>John</li>
  <li>Karl</li>
  <li>Brandon</li>
</ul>
<ul>
  <li>Glen</li>
  <li>Tane</li>
  <li>Ralph</li>
</ul>
  1. 在每个 ul 中查找第一个 li
$("ul li:first-child")

# 7.2 :first-of-type1.9+

概述： 选择所有相同的元素名称的第一个兄弟元素。

示例如下：

html代码

<div>
	<span>1</span>
	<span>2</span>
	<span>3</span>
</div>
  1. 在每一个匹配的div中查找第一个span
$("div:first-of-type")

# 7.3 :last-child

概述： 匹配最后一个子元素。

示例如下：

html代码

<ul>
  <li>John</li>
  <li>Karl</li>
  <li>Brandon</li>
</ul>
<ul>
  <li>Glen</li>
  <li>Tane</li>
  <li>Ralph</li>
</ul>
  1. 在每个 ul 中查找最后一个 li
$("ul li:last-child")

# 7.4 :last-of-type1.9+

概述： 选择的所有元素之间具有相同元素名称的最后一个兄弟元素。

示例如下：

html代码

<p>
	<span>1</span>
	<span>2</span>
	<span>3</span>
</p>
  1. 选择的所有元素之间具有相同元素名称的最后一个兄弟元素
$("p:last-of-type")

# 7.5 :nth-child

概述： 匹配其父元素下的第N个子或奇偶元素。

示例如下：

html代码

<ul>
  <li>John</li>
  <li>Karl</li>
  <li>Brandon</li>
</ul>
<ul>
  <li>Glen</li>
  <li>Tane</li>
  <li>Ralph</li>
</ul>
  1. 在每个 ul 查找第 2 个li
$("ul li:nth-child(2)")

# 7.6 :nth-last-child()1.9+

概述： 选择所有他们父元素的第n个子元素。计数从最后一个元素开始到第一个。

示例如下：

html代码

<ul>
	<li>1</li>
	<li>2</li>
	<li>3</li>
	<li>4</li>
</ul>
  1. 在每个匹配的ul中查找倒数第二个li
$("ul li:nth-last-child(2)");

# 7.7 :nth-last-of-type()1.9+

概述： 选择的所有他们的父级元素的第n个子元素，计数从最后一个元素到第一个。

示例如下：

html代码

<ul>
	<li>1</li>
	<li>2</li>
	<li>3</li>
	<li>4</li>
</ul>
  1. 在每个匹配的ul中查找倒数第二个li
$("ul li:nth-last-of-type(2)");

# 7.9 :only-child

概述： 选择同属于一个父元素之下，并且标签名相同的子元素中的第n个。

示例如下：

html代码

<div>
  <span>John</span>
  <b>Kim</b>
  <span>Adam</span>
  <b>Rafael</b>
  <span>Oleg</span>
</div>
<div>
  <b>Dave</b>
  <span>Ann</span>
</div>
<div>  
  <i><span>Maurice</span></i>
  <span>Richard</span>
  <span>Ralph</span>
  <span>Jason</span>
</div>
  1. 查找每个span，这个 span 是 其所有兄弟span元素中的第二个元素
$("span:nth-of-type(2)");

# 7.10 :only-of-type1.9+

概述： 如果某个元素是父元素中唯一的子元素，那将会被匹配。

示例如下：

html代码

<ul>
  <li>John</li>
  <li>Karl</li>
  <li>Brandon</li>
</ul>
<ul>
  <li>Glen</li>
</ul>
  1. 在 ul 中查找是唯一子元素的 li
$("ul li:only-child")
